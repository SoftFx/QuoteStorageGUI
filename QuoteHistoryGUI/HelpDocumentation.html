Quote History tool

Commands

Open

The Open button calls the Open dialog when it is possible to browse or choose from the list path to quote storage. Then after pressing the Open button in the dialog it tries to open this storage in GUI tool

Import

The Import button calls the Import dialog when it is possible to import all files or file matched by templates from another storage. Also import can be done in Console mode by calling exe file with -import or -i parameter with source and destination pathes. For more information use -help/-h parameter with executable from console

Upstream

The Upstream button calls the Upstream dialog when it is possible to perform generation ticks files from ticks level2 files and M1 files from ticks files.

The Ticks to M1 update for ticks obtained from level2-check box
